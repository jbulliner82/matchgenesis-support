How do I play Match Genesis?

Swipe to swap adjacent gems and match 3 or more of the same type. Fill the rescue meter before time runs out to save the character and complete the level!

What are special gems?

Match 4 gems to create a Line Blaster (clears a row or column). Match 5 gems for a Rainbow Bomb (clears all gems of one color). Create L or T shapes for Area Bombs (3x3 explosion).

How do I earn 3 stars?

Stars are based on your score. Reach 150% or more of the target score to earn 3 stars. Create combos and chain matches for higher scores!

Is an internet connection required?

No! Match Genesis works completely offline. The AI story generation feature is optional and requires your own API keys if you want to use it.

What is the AI story generation feature?

This is an optional feature that lets you generate unlimited unique rescue scenarios using AI. You'll need to provide your own OpenAI or Anthropic API key. The game includes 20 built-in levels that work without this feature.

How do I reset my progress?

Delete and reinstall the app to reset all progress. Your game data is stored locally on your device.

The game is running slowly. What can I do?

Try closing other apps running in the background and restarting Match Genesis. If the issue persists, try restarting your device.
